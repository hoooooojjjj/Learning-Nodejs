# 인프런 Do it! Node.js 프로그래밍 입문 강좌 레파지토리

## 공부한 내용

- common JS 모듈 시스템
  - 사용자 정의 모듈
  - 내장 모듈1(노드 코어 모듈) : path, fs, http 등
  - 내장 모듈2(글로벌 모듈 - require 없이 사용 가능) : console, '\_\_dirname', '\_\_filename' 등
  - npm 모듈
- express 프레임워크
  - 익스프레스를 통해 서버 만들고 실행
  - 익스프레스 라우팅(get,post,put,delete)
    - Thunder Client(Postman 대체) : 가상 클라이언트 요청
    - 라우트 파라미터
